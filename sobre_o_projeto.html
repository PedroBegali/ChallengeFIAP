Sobre o Projeto

Nosso projeto busca resolver o problema da descentralização das informações, e aumentar a coleta de dados!

Criaremos um site oficial para atender as chamadas em um único canal. A home do site será triangulada (História, Recursos, Objetivo)
- História: Quem somos, passar credibilidade
- Recursos: Ajuda voluntária, ajuda financeira
- Objetivo: Conectar como as pessoas que são o foco do projeto

No site as pessoas que buscam por atendimento poderão se cadastrar, informando os dados necessários para mapear os principais problemas dentários e as regiões mais afetadas.
Pensamos também em fornecer mais praticidade para estas pessoas, simplificando o cadastro, sendo feito por meio de um atendente, via ligação telefônica, e este atendente que irá de fato alimentar o sistema com as informações necessárias ficando para o beneficiário apenas aguardar a data da triagem e verificando no site, temos algumas ideias sobre criar um plano de assinatura com algumas vantagens, outras ideias de criar jogos tendo em vista a idade atendida pelo projeto da ONG.
